# 📚 Library Management System

A modern **Library Management System** built using **React.js and Firebase**.
This project allows users to manage books online with authentication and real-time database support.

---

## 🚀 Features

* 🔐 User Authentication (Login / Signup)
* 📚 View all books from database
* ➕ Add new books
* 🔍 Search books by title
* 📦 Issue books (decrease available copies)
* 🔄 Return books (increase available copies)
* 🔥 Real-time updates using Firebase Firestore

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend:** Firebase
* **Database:** Cloud Firestore
* **Authentication:** Firebase Auth

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository

```
git clone https://github.com/your-username/library-management-system.git
cd library-app
```

---

### 2️⃣ Install Dependencies

```
npm install
```

---

### 3️⃣ Run the Project

```
npm start
```

Then open:
👉 http://localhost:3000

---

## 🔥 Firebase Setup

1. Go to Firebase Console
2. Create a new project
3. Enable:

   * Authentication → Email/Password
   * Firestore Database
4. Copy Firebase config into `firebase.js`
